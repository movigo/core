Movigo.js

Ultralight JS library to animate your DOM elements. For the installation of the library read the README file on Github. For use cases go to Ceditor gists.

  • ✔ Light (~3kb)
  • ✔ Modern
  • ✔ Easy to use
‣

Functions

First function to call is always target, which takes a selector or a DOM element as input and return an object of functions. We can divide these functions into action functions and option functions (or simply actions and options). Every action or option return other actions and options, with a final function: animate, that plays the animation and return a promise (fulfilled at the and of animation). Click on `play` button at top left to play animations. 


    const target = movigo.target('div')

    // or:

    const element = document.querySelector('div')
    const target = movigo.target(element)

    target.duration(1) // Option function.

    target.width('100px') // Action function.

    const animation = target.width('200px').duration(1)

    // Create the animation and wait for the end.
    await animation.animate()
‣

Method chaining

As you can see Movigo use method chaining technique, and save the state of the animation in each function. This means that you can break the chain and create new chains with additional functions by building different animations. 


    const animation = target.duration(.5)

    await animation.width('100px').animate()
    await animation.height('100px').animate()
‣

Actions

Actions correspond to CSS properties and their values must be the same values as CSS properties. To see a full list of available actions you can use movigo.actions() function. You can also see a full list of action functions in the README file. 


    await target
        .width('80px').height('80px')
        .rotateZ('90deg').translateY('-100px')
        .borderRadius('100%')
        .animate()
‣

Options

Options allow you to set some properties of the animations, such as duration, speed curve of the transition effect, delays or loops. In particular, a loop can be infinite or determined by a number. To see a full list of available options you can use movigo.options() function. You can also see a full list of option functions in the README file. 


    await target
        .duration(.5).delay(.5).easing('ease-in-out').loop(2)
        .width('80px').height('80px')
        .rotateZ('180deg')
        .animate()

    // Infinite loop:
    // target.left('100px').loop().animate()
